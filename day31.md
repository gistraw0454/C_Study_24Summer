# 31강 - 함수예제 및 실습, 저장 기억 부류
## 함수 선언시 순서
1. 함수명을 생각
2. 매개변수 생각
3. 실행문장 작성
4. 리턴값 작성
5. 리턴타입 작성

## 함수의 사용
함수명(값)-> 리턴값이 있으면 통채로를 리턴값으로 보기./ 없으면 값으로 보기 x

### 함수를 사용하는 이유
1. 재사용, 특정성을 부여해서는 안됨.
2. 소스코드 간결화

### 리턴을 해야할때
- 사용하는 부분에 작업의 결과를 전달해야할때
- 함수 안에서 모든 작업이 끝나지 않을때

## 저장기억부류 (Storage class)
- stack
    - 지역변수, 매개변수 (직접 초기화) {}이후 해제
- heap
    - 동적메모리 (직접 초기화) free()로 해제
- data
    - 전역변수, 정적변수 (자동 초기화) 프로그램종료시 해제
