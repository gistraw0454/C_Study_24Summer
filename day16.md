# 16강 - 제어문 - 반목문 for문, 누적 연산자

## 대입연산자 (복합대입 연산자, 누적 연산자)
- +=, -=, *=,ㅣ=,%=,...
    ```
    int money = 10000;
    money -= 1000;
    printf("%d",money);
    ```
## 증감연산자
- ++, --
    - 전위형 : ++data 해당라인부터 바로 적용
    - 후위형 : data++ 다음라인부터 적용

## 2. 반복문
- for 문
    ``` 
    for (초기식;조건식;증감식){
            반복할 문장;
    }
    ```
