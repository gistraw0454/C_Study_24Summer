# 13강 - 삼항 연산자 심화 예제, 제어문 - 조건문 if 이론

## 제어문
1. 조건문
    - if 문 : 위의 조건식의 결과와 상관없이 무조건 검사한다.
    - else if 문 : 위의 조건이 참이 아니면 순차적으로 검사한다.
    - switch문 : if2.c 보면 너무 지저분.
    ```
    switch(변수명){
        case 값1:
            실행문장;
        case 값2:
            실행문장;
        ...
        default:
            실행문장;
    }
    ```
