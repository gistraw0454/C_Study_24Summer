# 32강 - 포인터와 목적
## 포인터 : 주소값
## 포인터 변수 : 주소를 담는 변수
## 포인터 상수 : 주소를 담는 상수 (배열)

- & : 주소 연산자 
- 자료형* : 포인터 연산자
- *주소값 : 참조 연산자

### 포인터를 사용하는 이유
1. 다른영역에 있는 변수의 값을 수정하기 위해
    - 함수는 한개의 값만 리턴 할 수 있으므로 포인터를 사용해값 변경
2. 동적 메모리 할당
    - `stdlib.h` 필요
    - 동적할당 (dynamic allocation)
        - 동적 배열할당
            - `자료형* 포인터명 = (자료형*)malloc(sizeof(자료형)*길이);`
        - 동적 배열 초기화 (clear allocation)
            - `자료형* 포인터명 = (자료형*)calloc(길이, sizeof(자료형));`
        - 동적 배열 크기 변경 (resize allocation)
            - `포인터명 = (자료형*)realloc(포인터명, sizeof(자료형)*길이);`
        - 동적 메모리 해제
            - `free(포인터명);`
            - 메모리를 해제하지 않으면 사라지지않고남아있기 떄문에 꼭 반드시 free함수로 메모리 해제를 해줘야함.
