# 30강 - 함수
- 함수명(매개변수) 리턴값
### 함수선언
```
리턴타입 함수명(자료형 매개변수1,..){
    실행문장;
    return 리턴값;
}
```
- 리턴값의 타입을 작성. 만약 리턴값이 없다면 비워놓지말고 void리턴
- 함수이름은 동사로작성
- 매개변수는 생략이 가능하지만, 외부에서 값을 전달받기 위해서는 반드시 자료형과 순서에 맞게 선언해줘야함.
- 리턴타입이 void면 리턴값 생략가능

### 함수 선언과 사용구분
{}가 있으면 선언, {}가 없으면 사용
### 함수 선언시 주의사항
- 함수 안에 다른 함수를 선언할 수 없으며, 함수 사용은 반드시 함수 안에서 사용해야함
- 함수를 사용할 때 선언부 소스코드 밑에 있으면 인식이 안되기 때문에, 반드시 함수 선언은 사용하는 부분 위에 선언 되어야한다.
